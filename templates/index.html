Gregg Shorthand Lookup is a small study applet for searching English words and phrases, displaying available Gregg shorthand outlines, and showing study metadata such as frequency, IPA, and the numerical value in the Major System.

This project is open source. View the source code on GitHub .

Cheat Sheet

Word Frequency Metric

The frequency score is derived from the Python wordfreq package. The backend reads a Zipf frequency value with zipf_frequency(word, "en"), then converts that value into this app's custom 0–100 study rating.

The displayed number is therefore not a strict dictionary rank or percentile. It is a practical study-weighting score: higher values indicate words that are more common in modern English usage.
